# Command: List Issues

Alias for `/list-work`.

Use `.cursor/commands/list-work.md` as the source of truth. In particular:

- Run `bd list` in each subdirectory with a `.beads/` directory.
- Group results by subdirectory.
- Mark any issue with the `recurring` label as `recurring`.
- Do not create, update, close, or commit issues from this command.
